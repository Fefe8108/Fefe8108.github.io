Password Manager

Les Fonctions de notre Password Manager

Notre password manageur demande a l'utilisateur de choisir son email et code pour rentrer dans l'application. Ensuite, dès qu'il rentre dans l'application, il va mettre les sites web avec les informations pour se connecter.

Qu'a-t-on utilisé pour le coder?

Nous avons utilisé Python et Repl.it afin de coder notre application.
